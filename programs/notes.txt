python - objects 
-----------------------

number 
-------
number can be integer or decimal or octal or hexa.

aval = 10 
bval = 34.3

string 
-------
string is defined in single or double or triple quotes.

aname = 'python programming'
bname = "java programming"

list ( array in c)
-----
list contains set of elements.
elements can be numbers or strings or any combination.
list is defined in [].

alist = [10,20,30,40]
blist = ["java","oracle"]
clist = [40,45.43,"java",["unix","oracle"]]


tuple ( immutable unchangeable uneditable)
-------
tuple contains set of elements.
elements can be numbers or strings or any combination.
tuple is defined in ().
** Elements inside tuple cannot be modified directly ***

atup = (10,20,30)
btup = ("java","oracle")


dictionary  ( json   hashmap   hash   map   table  linkedmap  treemap)
-----------
dictionary contains elements in keys and values.
dictionary is defined in {}.
In dictionary, keys are unique . and key can be either number or string to identify the value
value can be number , strin , list ,tuple , dictionary or any other complex datastructure.


syntax:  object = { key:value , key:value , key:value }
example:   book = {"chap1":10  ,"chap2": 20 ,"chap3":30 }

working with flat files 
-----------------------------

opening the file 
--------------------
syntax:   object = open(filename, mode)
example:    fobj = open("customers.txt","r")
            case 1: if the file is not doesnt exist .. it throws error.
            case 2: if the file is existing .. we will be able to read the file 

            fobj = open("customers.txt","w")
            case 1: if the file is not existing .. file gets created first 
            case 2: if the file is already existing .. it overwrites

            fobj = open("customers.txt","a")
            case 1: if the file is not existing .. file gets created first 
            case 2: if the file is already existing .. it adds the new content.




closing the file 
----------------
fobj.close()


reading the file 
-----------------------
1. reading the file line by line using file object 
2. fobj.readlines()
3. fobj.read()
4. using csv library 
5. using pandas library


writing to the file 
----------------------
1. fobj.write()
2. fobj.writelines()
3. using csv library
4. using pandas library
